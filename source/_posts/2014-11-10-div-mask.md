# 应用场景

在网站开发过程中，经常有层与层之间叠加的需求，比如假如用户没有登陆，则加一层蒙板显示“登陆后才可发表”：

<img src='/assets/images/div-mask-1.png'>

<!--break-->

# 代码解释

假设 id='content' 是我们的内容层， id='mask' 是半透明叠加层

## html

``` html
<div id='content'>
    <textarea> 加入讨论，帮助完善方案 </textarea>
    <button> 发 送 </button>
    <div id='mask'>
        <a id='hint'>登陆</a>后才能发表
    </div>
</div>
```
## css

``` css
#content {
     position: relative;
}

#mask {
    position: absolute;
    z-index: 1000;
    width: 100%;
    height: 100%;
    top: 0;
    left: 0;
    background-color: rgba(58, 58, 58, 0.2);
    text-align: center;
}
 
#mask #hint {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    -webkit-transform: translate(-50%, -50%);
    -moz-transform: translate(-50%, -50%);
    -ms-transform: translate(-50%, -50%);
    -o-transform: translate(-50%, -50%);
}
```

主要是 #content 和 #mask 的样式设置，#hint的设置主要为了居中显示
